## 技术栈推荐

### 前端 (Flutter)
```yaml
核心依赖:
  flutter_bloc: ^8.1    # 状态管理
  go_router: ^12.0      # 路由
  dio: ^5.3             # HTTP
  fl_chart: ^0.64       # 图表
  flutter_localizations: # 多语言
```

### 后端 (Rust)
```toml
核心依赖:
  axum = "0.7"          # Web框架
  tokio = "1.35"        # 异步运行时
  sqlx = "0.7"          # 数据库
  hdf5 = "0.8"          # 科学数据
  tokio-modbus = "0.9"  # Modbus协议
  jsonwebtoken = "9.0"  # 认证
```

### 数据库
- **SQLite**: 元数据（用户、配置、索引）
- **HDF5**: 科学数据（实验数据、波形）

## Release规划

### Release 1 (Sprint 1-8): 核心基础设施
✅ 包含：
- 用户/团队管理
- 仪器三层结构管理
- 虚拟设备支持
- 桌面端部署（Windows/macOS/Linux）
- 浅色/深色主题
- 中英文支持

❌ 不包含：
- 真实仪器协议
- Web部署
- Python客户端

### Release 2 (Sprint 9-18): 试验控制 + 数据管理
✅ 新增：
- Modbus协议
- CAN协议
- 试验方法编辑器
- 实时数据可视化
- 数据浏览器
- 容器化部署

❌ 不包含：
- VISA协议
- Python客户端

### Release 3 (Sprint 19-26): 高级功能
✅ 新增：
- VISA协议
- Python客户端库
- LaTeX图表导出
- 混合部署
- 法文支持
- 性能优化
