# Release 0 - 科学研究支持软件 Kayak

**版本**: 0.1.0  
**发布日期**: 2026-04-01  
**状态**: 初始发布

---

## 概述

Kayak 是一个科学研究支持软件，提供仪器管理、数据采集、试验控制和数据分析功能。本版本（Release 0）是初始版本，完成了项目基础框架和核心功能的开发。

## 主要功能

### ✅ 已完成功能

#### 1. 项目基础框架

- **Rust 后端**: 基于 Axum 框架的 REST API
- **Flutter 前端**: 跨平台桌面和 Web 应用
- **SQLite 数据库**: 轻量级数据存储
- **JWT 认证**: 安全的用户认证系统

#### 2. 用户认证

- 用户注册和登录
- JWT Token 管理
- 会话状态持久化
- 路由守卫

#### 3. 工作台管理

- 创建、编辑、删除工作台
- 工作台列表和详情查看
- 工作台与设备的关联

#### 4. 设备与测点管理

- 虚拟设备协议支持
- 设备树形结构
- 测点 CRUD 操作
- 测点值实时读取

#### 5. 试验数据管理

- 试验列表和详情
- HDF5 数据存储
- 测点历史数据查询
- 数据导出 (CSV)

#### 6. 部署支持

- Docker 单容器部署
- Docker Compose 多容器部署
- 本地开发脚本
- 桌面应用构建配置

- Rust 1.75+

#### 前端
- Flutter 3.16+
- 支持 Windows/macOS/Linux

## 快速开始

### Docker 部署

```bash
# 克隆项目
git clone https://github.com/hoolheart/kayak.git
cd kayak

# 启动服务
docker-compose up -d

# 访问应用
open http://localhost:8080
```

### 本地开发

```bash
# 启动后端
cd kayak-backend
cargo run

# 启动前端（新终端）
cd kayak-frontend
flutter run
```

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v1/auth/login` | POST | 用户登录 |
| `/api/v1/auth/register` | POST | 用户注册 |
| `/api/v1/auth/refresh` | POST | 刷新 Token |
| `/api/v1/workbenches` | GET/POST | 工作台列表/创建 |
| `/api/v1/workbenches/{id}` | GET/PUT/DELETE | 工作台详情/更新/删除 |
| `/api/v1/experiments` | GET | 试验列表 |
| `/api/v1/experiments/{id}` | GET | 试验详情 |
| `/api/v1/experiments/{id}/points/{channel}/history` | GET | 测点历史数据 |

详细 API 文档请参考 [docs/api.md](api.md)。

## 已知限制

1. **协议支持**: 仅支持虚拟设备协议（Virtual）
2. **数据存储**: 使用 SQLite，适合小规模数据
3. **用户管理**: 单用户模式，无团队功能
4. **试验控制**: 基础状态机，无复杂流程控制
