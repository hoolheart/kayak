| 会话过期 | 显示警告横幅 (Warning Container 背景) |
| 网络错误 | 显示错误横幅 "网络连接失败，请检查网络" |

| < 768px | 卡片全宽，左右 margin 16px，内边距 32px 24px |
| < 480px | 卡片内边距 24px 16px |

- 错误横幅: Error Container #B71C1C, On Error Container #FFEBEE
- 阴影: Elevation 1, 0px 2px 4px rgba(0,0,0,0.32)
