```
┌─────────────────────────────────────────────────────────────┐
│                                                             │

│                    │   科学研究支持平台 │  Body Medium         │
│                    │                 │                      │
│                    │  ┌───────────┐  │                      │
│                    │  │ 邮箱      │  │  Input               │
│                    │  └───────────┘  │                      │
│                    │  ┌───────────┐  │                      │
│                    │  │ 密码      │  │  Input               │
│                    │  └───────────┘  │                      │
│                    │                 │                      │
│                    │  [  登 录  ]    │  Primary Button      │

│                    │                 │                      │

│                          max-width: 400px                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 视觉规格

| 元素 | 规格 |
|------|------|
| 页面背景 | Surface color (浅色: #FFFFFF, 深色: #0F172A) |
| 卡片背景 | Surface Container Low (浅色: #F1F5F9, 深色: #1E293B) |
| 卡片圆角 | 28px |
| 卡片内边距 | 32px |
| 卡片阴影 | Elevation 2 (浅色), Elevation 1 (深色) |
| Logo 图标 | science icon, 48px, Primary color |
| 标题 "Kayak" | Headline Small, On Surface, 居中 |
| 副标题 | Body Medium, On Surface Variant, 居中 |
| 输入框间距 | 16px |
| 按钮上边距 | 24px |
| 注册链接上边距 | 16px |

### 1.3 交互规格

| 状态 | 表现 |
|------|------|
| 初始状态 | 邮箱输入框自动聚焦 |
| 输入验证 | 实时验证邮箱格式，密码长度 >=6 |
| 登录中 | 按钮显示加载指示器，禁用输入 |
| 登录成功 | 页面淡出，跳转 Dashboard |
| 登录失败 | 显示错误提示横幅（Error Container 背景） |
| 会话过期 | 显示警告提示（Warning Container 背景） |

### 1.4 响应式

| 断点 | 调整 |
|------|------|
| >=768px | 卡片居中，max-width 400px |
| <768px | 卡片全宽，padding 16px |
