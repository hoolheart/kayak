## 1. 测试概述

### 1.1 测试目标

验证基于新 Figma 设计重构的 4 个核心页面的 UI 正确性，确保：

- **登录页**：440px 居中卡片布局、品牌元素渲染、表单输入验证、错误/加载状态、密码可见性切换
- **Dashboard**：欢迎区域动态问候语、4个快捷操作卡片、统计数字展示、最近工作台列表、空状态设计
- **工作台列表页**：网格/列表双视图切换、搜索筛选功能、CRUD 对话框、空状态与加载骨架屏
- **工作台详情页**：设备树面板（280px固定宽度）、Tab 导航（设备列表/设置）、设备卡片展开/折叠、连接状态、测点值展示

### 1.2 关键设计约束

| 约束 | 描述 | 参考 |
|------|------|------|
| Web 模式优先 | 测试以 Flutter Web 为主要目标平台 | 任务要求 |
| 浅色/深色双主题 | 所有页面必须支持 Light/Dark 主题切换 | Figma 2.1, 6.x |
| Material 3 Design | 使用 M3 组件体系（Filled fields, surface colors, tokens） | Figma 3.x |
| 响应式布局 | Desktop(≥1280), Tablet(≥768), Mobile(<768) | Figma 7.x |
| 卡片大圆角风格 | 卡片圆角 12px-28px 统一风格 | Figma 3.x |

### 1.3 涉及的页面

| 页面 | 路由 | Figma 参考 |
|------|------|------------|
| 登录页 | `/login` | `log/release_1/ui/figma/login_page.md` |
| Dashboard | `/dashboard` | `log/release_1/ui/figma/dashboard_page.md` |
| 工作台列表 | `/workbenches` | `log/release_1/ui/figma/workbench_list_page.md` |
| 工作台详情 | `/workbench/:id` | `log/release_1/ui/figma/workbench_detail_page.md` |

### 1.4 测试策略

- **Widget 测试**：使用 `flutter_test` 和 `WidgetTester`，验证组件渲染、交互状态、条件显示
- **视觉回归测试**：截图与 Figma 原型对比，验证颜色、间距、字体、阴影等视觉属性
- **集成测试**：完整用户旅程验证（登录→Dashboard→工作台列表→工作台详情）
- **手动探索性测试**：人模拟操作，测试实际浏览器交互体验
- **响应式测试**：多视口尺寸下的布局和可读性验证

### 1.5 测试范围边界

| 测试维度 | 覆盖 | 不覆盖 |
|----------|------|--------|
| UI 渲染 | 组件存在性、位置、尺寸、颜色、文本 | 后端 API 逻辑 |
| 用户交互 | 点击、输入、切换、导航 | 数据持久化逻辑 |
| 状态转换 | Loading、Empty、Error、Success | 网络层异常处理 |
| 主题切换 | Light↔Dark 视觉变化 | 系统级主题同步 |
| 响应式 | 3 个断点布局变化 | 横竖屏切换（非 Web 重点） |
| 表单验证 | 前端即时验证规则 | 后端业务验证 |

### TC-LOG-001: 登录卡片居中显示（440px 宽度）

| 字段 | 内容 |
|------|------|
| **测试ID** | TC-LOG-001 |
| **测试名称** | 登录卡片居中显示（440px 宽度） |
| **测试目的** | 验证登录卡片在视口中水平垂直居中，宽度为 440px |
| **前置条件** | 登录页已实现，桌面视口（1440×900） |
| **测试步骤** | 1. 导航至 `/login` 路由<br>2. 使用 WidgetTester 定位 LoginCard 组件<br>3. 验证卡片在视口中水平居中<br>4. 验证卡片宽度为 440px（或等价逻辑像素）<br>5. 验证卡片 padding 为 40px 32px<br>6. 验证卡片圆角为 28px |
| **预期结果** | 1. 卡片位于视口中央（mainAxis: center, crossAxis: center）<br>2. 卡片约束宽度 ≈ 440px<br>3. 圆角半径 = 28px<br>4. padding 为 EdgeInsets.symmetric(horizontal: 32, vertical: 40) |
| **测试数据** | 桌面视口 1440×900 |
| **优先级** | P0 |

**Flutter 测试代码示例**：
```dart
testWidgets('login card centered with 440px width', (tester) async {
  await tester.pumpWidget(const KayakApp(initialRoute: '/login'));
  await tester.pumpAndSettle();

  // Find the login card by its constrained width or semantic label
  final cardFinder = find.byType(LoginCard); // or by key
  expect(cardFinder, findsOneWidget);

  // Verify centering
  final cardRect = tester.getRect(cardFinder);
  final viewSize = tester.view.physicalSize / tester.view.devicePixelRatio;
  expect(cardRect.center.dx, closeTo(viewSize.width / 2, 1.0));
  expect(cardRect.center.dy, closeTo(viewSize.height / 2, 1.0));

  // Verify card width (440px logical)
  expect(cardRect.width, closeTo(440, 2.0));
});
```
