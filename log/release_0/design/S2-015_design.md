# S2-015 详细设计文档：Dashboard首页

**任务名称**: Dashboard首页
**创建日期**: 2026-04-04
**版本**: 1.0

---

## 1. 任务概述

### 1.1 目标
实现Dashboard首页，包含：
1. 快捷操作入口(创建工作台、启动试验)
2. 最近工作台列表
3. 最近试验列表
4. 系统状态概览

### 1.2 验收标准
- [ ] 快捷入口可点击跳转
- [ ] 最近列表展示正确
- [ ] 空状态提示友好

---

## 2. 架构设计

### 2.1 页面结构

```mermaid
graph TB
    A[DashboardPage] --> B[快捷操作区]
    A --> C[最近工作台列表]
    A --> D[最近试验列表]
    A --> E[系统状态概览]
    
    B --> B1[创建工作台按钮]
    B --> B2[启动试验按钮]
    
    C --> C1[工作台卡片列表]
    C --> C2[查看全部链接]
    
    D --> D1[试验状态列表]
    D --> D2[查看全部链接]
    
    E --> E1[设备连接状态]
    E --> E2[系统资源使用]
```

### 2.2 数据流

```mermaid
sequenceDiagram
    participant U as User
    participant D as DashboardPage
    participant W as WorkbenchService
    participant E as ExperimentService
    
    U->>D: 打开Dashboard
    D->>W: 获取最近工作台
    D->>E: 获取最近试验
    W-->>D: 返回工作台列表
    E-->>D: 返回试验列表
    D-->>U: 渲染Dashboard
```
